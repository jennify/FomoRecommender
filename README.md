# FomoRecommender

## Setup

###To run:
<pre><code>
python manage.py runserver
</code></pre>

###To migrate db:

<pre><code>
python manage.py migrate
python manage.py makemigrations
</code></pre>

### Database cleared:

A. Delete all tables

python manage.py sqlclear

B. delete all data in all tables

python manage.py flush
